HTML

  • HTML stands for Hypertext Markup Language.
  • It is used to give your website, or content, structure; like the framing of a house.
  • The head element contains information about the webpage and is not always visible to the end user.
  • There are several different metadata tags that can be used within the head element such as Title to denote the title of the webpage or Link which can link an external resource to your webpage.
  • The body element represents the visible content shown to the user.
  • Content for visible HTML elements are contained between and opening tag and a closing tag. Each tag is represented by angle brackets and the tag name listed inside them.

CSS

  • Cascading Style Sheets
  • Used to define how webpage content is presented to the end user.
  • Three different style methods; Inline CSS, Internal CSS Style Sheet, External CSS Style Sheet.
  • Inline CSS; used to style one specific element directly. Quick but not preferable for large changes.
  • Internal CSS; used to style webpage in the head section of HTML. Changes made to this rule will apply to all tagged elements.
  • External CSS: Most common method and preferred because you can store all rules in an external file which you can then link to your website.
  • Declarations contain CSS property to be worked with and the value to apply to that property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * Declaration indicates you want ALL the elements on that webpage to contain the following properties.

Git

  • Global Information Tracker
  • Version control system that can have multiple developers working on the same project at the same time.
  • Helps ensure that any changes made by one developer does not affect another developers work.
  • cd: change directory.
  • ls: lists the files and directories.
  • pwd: print working directory, will show what directory you are currently in.
  • mkdir directory-name: makes a new directory.
  • touch file-name: creates a new file within the working directory.
  • start file-name: opens the file.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is used to power websites with interactivity.
  • A variable is a named container that allows us to store data in our code.
  • Can declare a variable by using the "let" keyword, followed by the variable name.
  • There are 5 different data types that can be used; String, Number, Boolean, Array, and Object.
  • String data types is a sequence of text enclosed in single or double quotation marks.
  • Number data types is exactly what it sounds like, numbers.
  • Boolean data types depicts true or false values.
  • Array data types allow you to store multiple values in a single reference.
  • Object data types can encompass everything.
  • Comments are depicted by /# text #/ and do not show up in your code.
  • Conditionals are code structures used to test in an expression results as true or not.
  • Examples of conditionals include if statements and if...else statements.
  • Functions are a method used to package a block of code that performs a specific function that we can re-use.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript reads the code from top to bottom.
